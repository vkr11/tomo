# Company Research Template

> Copy this folder structure for each target company.

## Folder Structure

```
companies/<company_name>/
├── README.md                    # Overview, cross-refs, prep checklist
├── culture_principles.md        # Company values / leadership principles
├── interview_process.md         # Loop structure, round types, tips
├── team_research.md             # Specific team/org you're targeting
├── sample_questions.md          # Company-specific behavioral questions
└── _raw/                        # Web dumps, raw articles, unformatted references
```

## What to Research Per Company

### 1. Culture & Principles (culture_principles.md)
- [ ] Official values/principles (with exact wording)
- [ ] Interview signal per principle (what does "strong" look like?)
- [ ] Priority tiers (which principles are tested most?)
- [ ] Map your existing stories to their specific principles

### 2. Interview Process (interview_process.md)
- [ ] Number of rounds, format (phone/video/onsite)
- [ ] Round-by-round breakdown (who, what, how long)
- [ ] Special rounds (Bar Raiser, cross-functional, presentation)
- [ ] Written/take-home components
- [ ] Communication style expectations (BLUF vs. narrative, "I" vs. "we")

### 3. Team Research (team_research.md)
- [ ] What the team/org does and its strategic priority
- [ ] Recent launches, press releases, earnings mentions
- [ ] Key metrics the team owns
- [ ] Competitive landscape for that product area
- [ ] Who you'll report to (LinkedIn research)

### 4. Sample Questions (sample_questions.md)
- [ ] Behavioral questions mapped to their principles
- [ ] Product sense questions specific to their domain
- [ ] "Why this company?" — your authentic answer
- [ ] Questions YOU will ask them (team, culture, growth)
